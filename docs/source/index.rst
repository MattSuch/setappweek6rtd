Welcome to Speakeasy's documentation!
===================================

Speakeasy is a mobile application that specalises in teaching the user spanish through fun and exciting minigames. It presents this information in a *FUN* and *INTERESTING* way to help you learn in the best way possible.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
